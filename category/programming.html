Multithreading on python

Thu 02 January 2014

tags: python

Many times we need a concurrent execution of our programs. This is just a reminder of how to use multithreading in python.

#!/usr/bin/env python

    import threading, time
    count = 0


    def adder():
            global count
            count = count + 1
            time.sleep(0.5)
            count = count + 1

    threads = []
    for i in range(100):
            thread = threading.Thread(target=adder, args=())
            thread.start()
            threads.append(thread)

    for thread in threads: thread.join()
    print(count)

This program is quite simple and it's a quick reminder. It creates one hundred threads and uses them for increment a global variable.
